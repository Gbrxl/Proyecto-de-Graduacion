# Proyecto-de-Graduacion
Este proyecto de graduación será para tener una guía de como contrarrestas las medidas del Cambio Climático.
## Esta es la idea de mi proyecto:
> Crear una app de retos sostenibles de 30 días que guíe a las personas a cambiar sus hábitos diarios con acciones concretas y medibles. Cada reto completado muestra el Co ahorrado, generando conciencia y motivación a través de logros y rachas, convirtiendo el cuidado del planeta en un hábito diario.
> El proyecto será una página web construida con Flask y Python, usando HTML, CSS y plantillas Jinja.
